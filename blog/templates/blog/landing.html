About

Currently working as a System's Analyst for a large healthcare company, I made this website to serve as a place where I could store and write about projects I'm working on. At the time of writing, the projects tend to focus on supply chain and logistics. Mostly improving the flow of communication / data between users and identifying areas to improve in the supply chain.

Some of the tools I use in my projects:

  • Python (Lots of Python. Almost every project starts out as a Django app)
  • Javascript (JQuery, Chartjs, React)
  • AWS (RDS, S3, EC2, Lambda)
  • PostgreSQL (Big fan)

If that sounds interesting, you can read more about the projects Here.

I'm also keeping a blog on this site to write about the ecosystem of tools I use, and find interesting. It's hard to keep up with the incredibly fast pace of software, and I find that writing about something is a great way to learn the ins and outs.

If that sounds interesting, you can view the blog Here.
